# esp32_screwTerminalPCB
A screw terminal PCB for esp32 devkit

It's a 2 layer PCB.

Measures 2.30' x 3.60'

I used [easyEDA](https://easyeda.com/) to design it and ordered them through [jlcpcb](https://jlcpcb.com/)
